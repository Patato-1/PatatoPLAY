## 📚 学习顺序（严格按这个来）

### 阶段一：地基（W1-W4）
1. Python 基础 → 进阶 → 文件处理
2. Git 版本控制
3. 命令行工具
4. **第一个 AI 项目**：命令行翻译工具

### 阶段二：LLM 核心（W5-W7）
5. DeepSeek API 调用
6. Prompt 工程（5 大技巧）
7. 阶段项目（周报生成器、AI 面试官）

### 阶段三：核心技能（W8-W10）⭐
8. Embedding + Chroma 向量库
9. **RAG 完整系统**（简历项目 1）
10. **Agent + Function Calling**（简历项目 2）

### 阶段四：包装（W11-W12）
11. FastAPI 包装 HTTP 接口
12. Streamlit 做简单前端
13. 作品集打磨 + 演示视频

### 阶段五：求职（W13-W14）
14. 简历 + 面试题
15. 海投 + 面试

## 🎓 每个阶段要掌握到什么程度？

### W1-W4 地基
- 能独立写 200-300 行 Python 程序
- 熟练用 Git 提交代码
- 能调用外部 API

### W5-W7 LLM
- 熟练调用 DeepSeek API
- 掌握 5 种 Prompt 技巧
- 做出 2-3 个能用的小工具

### W8-W10 核心 ⭐
- 理解 Embedding 和向量检索原理
- 能搭完整 RAG 系统（文档→切片→检索→生成）
- 能用 Agent 调用多个工具

### W11-W12 包装
- 把项目包装成 HTTP 接口
- 用 Streamlit 做简单前端
- 项目有 README 和演示视频

### W13-W14 求职
- 简历能讲清项目经验
- 能回答 10 个常见面试题
- 投递 50+ 公司

## ❌ 不要学的东西

- ❌ Transformer 数学推导（应用层用不到）
- ❌ 模型训练（不是应用开发）
- ❌ React/Vue 前端（用 Streamlit 替代）
- ❌ Docker/K8s（先本地跑通）
- ❌ MySQL（先用 SQLite + 文件存储）
- ❌ 多框架对比（只学 LangChain）

## 💡 学习方法论

1. **3-2-1 法则**：每个新概念
   - 3 个例子（看懂）
   - 2 个练习（会做）
   - 1 个项目（用上）

2. **费曼技巧**：学完一个概念，假装给小白讲一遍

3. **番茄工作法**：25 分钟学 + 5 分钟休息

4. **教学相长**：每周末写一篇学习笔记（讲给自己听）
