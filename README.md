# aiday1

每天一个小项目，记录成长。

## Day 1 · 番茄钟

极简风格的网页番茄钟，含下班提醒功能。

**功能**

- 番茄钟计时：开始 / 暂停 / 重置
- 圆环进度动画（25 分钟工作）
- 每天 19:00 下班提醒弹窗
  - **确定**：当天不再提醒
  - **取消**：半小时后再提醒一次
- 调试按钮（右下角），方便随时测试弹窗
- 状态用 `localStorage` 持久化

**运行方式**

直接双击 `day1/番茄钟/index.html`，在浏览器中打开即可。

```text
aiday1/
├── README.md
└── day1/
    └── 番茄钟/
        └── index.html
```
